🚀 Deployment Options

1️⃣ Render (Recommended)

  1. Go to Render Dashboard
  2. Click "New+" → "Web Service"
  3. Configure:
    • Name: your-bot-name
    • Region: Singapore
    • Select option: Existing image
    • Plan: Free
  4. Add environment variables
  5. Deploy!

2️⃣ Koyeb (Alternative)

  1. Sign up at Koyeb
  2. Create App → "Deploy from Existing image"
  3. Use image: docker.io/woodcraftbot/1tamilmv-bot:latest
  4. Add env variables
  5. Deploy
